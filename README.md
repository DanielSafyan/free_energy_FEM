# 1D Fickian Diffusion Simulation using FEM

This project simulates 1D Fickian diffusion using the Finite Element Method (FEM) in Python.

## Dependencies

- NumPy
- SciPy
- Matplotlib

## How to Run

1. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the simulation:
   ```bash
   python fickian_diffusion_fem.py
   ```

The script will generate a plot (`diffusion_profile.png`) showing the concentration profile at different time steps.
